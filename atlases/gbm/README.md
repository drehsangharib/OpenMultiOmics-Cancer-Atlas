# GBM Atlas Module

The GBM module is the first flagship atlas in OpenMultiOmics-Cancer-Atlas.

## Public-data-only policy

This module must use only public data and synthetic/demo examples. Unpublished, private, or lab-internal GBM data must remain outside this repository.

## Priority biology

- GBM tumor-intrinsic proliferation/invasion
- PI3K-AKT, MAPK, STAT, NF-kB signaling
- tumor microenvironment
- microglia/macrophage activation
- astrocyte reactivity
- oligodendrocyte/myelination programs
- neuronal/synaptic brain-context signatures
- human tumor vs mouse model/context labeling
