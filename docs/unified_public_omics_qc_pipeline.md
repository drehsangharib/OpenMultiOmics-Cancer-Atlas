\# Unified Public Omics QC Pipeline



\## Purpose



`core/pipelines/run\_unified\_public\_omics\_qc\_pipeline.py` runs the unified public cancer omics metadata workflow and generates the unified inventory QC report in one command.



This module is part of Milestone 1F.6: unified public omics QC pipeline wrapper.



\## Scope



This workflow is metadata-only.



It does not download:



```text

expression matrices

copy-number matrices

methylation matrices

mutation matrices

clinical matrices

large molecular files
