\# Keyword-Driven Public Omics Atlas Builder



\## Purpose



`core/atlas/build\_keyword\_public\_omics\_atlas.py` builds a reusable metadata-only atlas slice from the unified public cancer omics inventory using keyword matching.



This module generalizes the disease-slice concept so that atlas slices such as GBM, LGG, LUAD, BRCA, PDAC, or CRC can be created from the same workflow pattern.
