# GDC Project Priority Report

## Purpose

`core/reporting/gdc_priority_report.py` generates an HTML report summarizing GDC project metadata, modality coverage, project priority rankings, and optional static visual outputs.

This module is part of Milestone 1C: local reporting and audience-facing summary outputs.
