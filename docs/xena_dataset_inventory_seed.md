\# UCSC Xena Dataset Inventory Seed



\## Purpose



`core/search/xena\_dataset\_inventory\_seed.py` creates a curated seed inventory of high-value UCSC Xena dataset categories relevant to cancer multi-omics atlas construction.



This module is part of Milestone 1E.3: UCSC Xena dataset inventory seed.



This is not a live Xena crawler. It defines the first dataset categories and matrix families that should be targeted by future Xena dataset crawling and integration modules.
