\# Cross-Atlas Summary Report



\## Purpose



`core/atlas/report\_cross\_atlas\_summary.py` generates a framework-level comparison report across multiple atlas slices.



This report is intended to summarize atlas coverage at a glance.



It compares atlas-level metrics such as:



```text

total rows by atlas

source coverage by atlas

modality coverage by atlas

unknown modality counts by atlas

missing source URL counts by atlas

QC HTML size by atlas
